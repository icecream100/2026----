

| 模块                | 一级功能          | 功能描述                                                                  | 备注                               |
| :---------------- | :------------ | :-------------------------------------------------------------------- | :------------------------------- |
| **物联网设备管理** (103) | **平台设备管理**    | 负责接入网关及下级平台的生命周期管理，包括接入网关的注册、编辑、删除以及接入状态的实时监控。                        | 复用视频云平台的下级平台级联-平台设备管理功能          |
|                   | **平台通道管理**    | 管理从下级平台同步的设备通道资源，支持查看通道树、手动同步目录以及管理设备的业务上架/下架状态。                      | 复用视频云平台的下级平台级联-平台通道管理功能          |
|                   | **设备分组管理**    | 维护设备分组字典，提供分组的增删改查功能，用于设备归类管理。                                        | 复用视频云平台的数据治理-设备类别管理              |
| **设备空间管理** (101)  | **基础设备目录管理**  | 统一管理全量设备的基础信息与目录结构。支持设备的查看、编辑、数据重置（恢复原始值）以及组织架构的管理与同步。                | 复用视频云平台的设备管理-基础目录设备管理功能          |
|                   | **标签管理**      | 建立标签体系以灵活组织设备。支持标签的创建、维护以及设备与标签的批量关联（穿梭框操作）。                          | 定制化开发                            |
|                   | **地图管理**      | 即地图配置管理。支持创建和维护自定义地图，配置行政区划范围、关联目录及初始经纬度。                             | 复用视频云平台的电子地图-地图管理功能              |
|                   | **地图广场**      | 基于 GIS 地图的可视化展示与交互。提供设备点位的聚合展示、框选/圈选、搜索筛选及收藏功能。                       | 复用视频云平台的电子地图-地图广场功能              |
| **设备治理** (102)    | **治理任务**      | 治理流程的发起与管理。支持创建治理任务、划定待治理设备范围并派发给指定的治理人员。                             | 复用视频云平台的数据治理-治理任务功能              |
|                   | **治理参数设置**    | 维护设备治理所需的标准化字典参数，如场所类型、地理区域等枚举值的配置。                                   | 复用 视频云平台的系统设置-数据治理参数配置           |
| **视频权限管理** (104)  | **直接共享 (应用)** | 管理第三方应用的接入授权。支持应用信息的注册维护、上架管理以及为应用分配设备访问权限。                           | 复用视频云平台的应用共享-应用管理功能，增加设备授权功能     |
|                   | **代理共享 (上级)** | 管理通过 GB28181 协议级联的上级监管平台。支持配置级联参数（SIP 信息）、注册状态监控及目录/设备的批量共享授权。        | 复用视频云平台的上级平台级联-上级平台管理功能，增加设备授权功能 |
| **权限审批管理** (105)  | **需求登记**      | 面向开发者的权限申请入口。支持发起按设备或标签的权限申请（含维度与期限），追踪申请单状态，查看详情与授权凭证，并支持申请撤回（二次确认）。 | 定制化开发                            |
|                   | **需求审批**      | 管理员对权限申请的审核与管控。支持待办/已办任务分标签管理，对申请单进行通过/驳回操作，调整授权范围/期限，以及查看历史审批详情。     | 定制化开发                            |
|                   | **授权管理**      | 监控当前生效的授权关系。支持按应用/状态筛选，查看授权详情（含Token及设备清单），并支持对生效权限进行强制撤销（二次确认）。      | 定制化开发                            |
